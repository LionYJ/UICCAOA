Organizer for UICC APDU Command

Supported UICC APDU COMMANDS
SELECT Support Select File
CREATE FILE This function allows the creation of a new file under the current directory.
DELETE FILE This command initiates the deletion of a referenced EF immediately under the current DF, or a DF with its complete subtree.
DEACTIVATE FILE & INVALIDATE This function initiates a reversible deactivation of an EF.
ACTIVATE FILE & REHABILITATE This function reactivates a deactivated EF
TERMINATE FILE & CARD USAGE The TERMINATE DF command initiates the irreversible transition of the currently selected DF/ADF,EF,UICC into the termination state.
RESIZE FILE This command allows to modify the memory space allocated to the MF, a DF/ADF, a transparent file or a linear fixed file or a BER TLV structured EF under the current directory (MF, DF/ADF).
READ BINARY This function reads a string of bytes from the current transparent EF.
UPDATE BINARY This function updates the current transparent EF with a string of bytes.
READ RECORD This function reads one complete record in the current linear fixed or cyclic EF.
UPDATE RECORD This function updates one specific, complete record in the current linear fixed or cyclic EF.
SEARCH RECORD & SEEK This function searches through a linear fixed or cyclic EF to find record(s) containing a specific pattern.
INCREASE This function adds the value given by the terminal to the value of the last increased/updated record of the current cyclic EF, and stores the result into the oldest record.
RETRIEVE DATA
SET DATA
VERIFY PIN This function initiates the comparison in the UICC of the PIN verification data sent from the terminal with the PIN reference data stored in the card.
CHANGE PIN The Change PIN command is used to initiate the comparison of the verification data with the PIN, and then to conditionally replace the existing PIN with the new PIN sent to the UICC in the command.
DISABLE PIN The Disable PIN command is used to switch off the requirement to compare the PIN verification data with the PIN reference data.
ENABLE PIN The Enable PIN command is used to switch on the requirement to compare the PIN verification data with the PIN reference data.
UNBLOCK PIN This function is used to reset the PIN retry counter to its initial value and then to conditionally set a new PIN value.
AUTHENTICATE
GET CHALLENGE This function is used to create a random number.
GET RESPONSE The command is used to transmit APDUs from the card to the terminal, which otherwise could not be transferred by the protocol.
MANAGE CHANNEL This command opens and closes logical channels.
MANAGE SECURE CHANNE
STATUS This function returns information concerning the current directory or current application.
TERMINAL CAPABILITY
TRANSACT DATA
TERMINAL PROFILE
ENVELOPE
FETCH This function is used to transfer a proactive command from the UICC to the terminal (e.g. from a CAT application).
